## 二、貓咖定義

### 📌 核心定義

> **貓咪咖啡廳（貓咖）**，是一種以「貓咪共存」為核心體驗的餐飲空間。
> 顧客在享用餐飲的同時，可與店內貓咪自由互動。
> 在台灣，貓咖多兼具「動物公益」使命——店內的貓通常來自救援、中途或棄養，
> 貓咖不只是吃喝的場所，更是**貓與人的庇護所、領養媒合的橋梁**。

### 三大識別要素（台灣語境）

> **① 貓咪在場且自由活動** × **② 貓有名字有故事（多為救援/中途）** × **③ 消費與動物公益連結**

### 空間特質

| 特質 | 說明 |
|------|------|
| 貓咪自由活動 | 不關籠，在店內遊走、趴睡、與人互動 |
| 多貓環境 | 通常 5–20 隻以上，各有名字與故事 |
| 貓友善設計 | 設有貓跳台、休憩區、隱藏貓砂盆 |
| 主題化空間 | 部分以特定品種/毛色為主題（如純黑貓、緬因貓） |
| 氣味管理 | 無異味是核心衛生評分指標 |

### 體驗特質

| 特質 | 說明 |
|------|------|
| 互動是主角，餐飲是媒介 | 核心動機是「與貓互動」，餐飲讓人停留 |
| 尊重貓咪節奏 | 強調讓貓自己選擇是否靠近 |
| 療癒氛圍 | 空間設計、燈光、音樂共創放鬆感 |
| 情感連結 | 每隻貓有名字與背景故事 |
| 限時制度 | 通常有用餐時限（1.5–3 小時）管控品質 |

### 商業模式特質

| 特質 | 說明 |
|------|------|
| 餐飲為主、動物公益為魂 | 收入來自餐飲，靈魂是救援使命 |
| 中途之家功能 | 店內貓多為「中途貓」，等待永久家庭 |
| 消費即贊助 | 「用一份餐點支持等待家的貓」 |
| 社群認同 | 吸引貓奴、動保支持者形成忠誠社群 |

### 3.5 經營者視角關鍵洞察

從 Threads 上的經營者貼文中，提取出以下核心矛盾：

```
「中途貓咖的根本困境」

收入：餐飲低消（200–550元）× 有限座位 × 有限翻桌率
支出：房租 + 人力 + 飼料 + 醫療費（萬起跳/次） + 中途照護
結果：損益兩平已是奢求，多數在賠錢狀態下撐著
出路：場地租借、周邊商品、贊助、認養媒合費、定期會員制
```

**最核心的經營矛盾：**
> 「憑什麼真心為貓好的人只能一直賠錢，憑什麼愛貓做公益的人不值得拿好薪水？」
> — @crazy.nekokafe（Threads 留言，94 讚）

## 四、游擊訪談受訪者條件

### 4.1 篩選標準

#### ✅ 必要條件（全部符合才訪）
1. 目前正在或剛離開貓咖（當下現場訪談）
2. 曾消費過至少 1 次貓咖（不限本店）
3. 願意接受 2–3 分鐘的簡短訪談

#### ⭐ 加分條件（有其中 2 項即為優質受訪者）
- 去過**超過 2 間**不同的貓咖
- 有固定回訪的貓咖（熟客）
- 曾在社群（IG/Threads/小紅書）分享過貓咖體驗
- 有養貓 → 對貓咪福利有意見
- 第一次來這間店（新鮮的第一印象）
- 帶朋友/伴侶/小孩來（社交型消費者）

#### ❌ 排除條件
- 店員、老闆、工讀生（非消費者身份）
- 完全沒去過貓咖（路人）
- 明顯趕時間、不願停下來的人
- 12 歲以下兒童（家長代答即可）

### 4.2 目標受訪者輪廓（根據 Threads 分析）

| 輪廓 | 描述 | 佔比（估計） |
|------|------|------------|
| 🐱 **療癒型** | 下班/下課後來擼貓減壓，無特定品種偏好 | ~40% |
| 📸 **打卡型** | 第一次來、拍照分享 IG，重視視覺體驗 | ~25% |
| 💼 **辦公型** | 帶電腦來邊工作邊吸貓，重視不限時 | ~15% |
| 🐾 **公益型** | 支持動保使命，關心貓咪來源與領養 | ~10% |
| 🐈 **品種控** | 特別為了特定品種（緬因/黑貓/布偶）來訪 | ~10% |

### 4.3 建議訪談時段與地點

| 建議 | 說明 |
|------|------|
| **最佳時段** | 平日下午 2–5 點（顧客較悠閒，不趕時間） |
| **次佳時段** | 假日上午開店後 1 小時（第一波客人剛入座後） |
| **避開時段** | 用餐高峰（12–1 點）、快限時前 30 分鐘（顧客有壓力） |
| **訪談位置** | 出口處（顧客剛結束體驗，感受最新鮮）或等待區 |
